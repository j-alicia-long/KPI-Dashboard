# KPI Dashboard
A KPI Dashboard built with Django, PostgreSQL, and Docker.

### Development Setup

Clone the project and run the docker container:
```
git clone [url]
cd KPI-Dashboard
docker-compose up
```
The Django server runs from the `web` service while PostgreSQL runs from `db`.
